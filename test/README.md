# iBuy SDK demo

## About the project

This project is for [iBuy SDK](https://github.com/dh1ki/ibuy-sdk) demonstration

## Getting Started

### Installation

`npm install` or `yarn install`

this will install the dependencies for running the demo script

### Prerequisites

Modify `config.ts.template` to fit your configuration, and rename it to `config.ts`

- `<YOUR_WALLET_SECRET_KEY>`: replace to your own one
- `<YOUR_RPC_URL>`: replace to your prefer one
- `<YOUR_MINT_SECRET_KEY>`: replace to your own one
- `<MINT_ADDRESS_FOR_TESTING>`: replace to your own one

### Usage

- `npm test <SCRIPT_NAME>` run the specific demo script, e.g. npm test create.ts.
